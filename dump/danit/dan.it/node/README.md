# fe-20-node-server

Node.js application + heroku integration 

## Required parts:
- heroku  - сервис для хостинга Node.js (heroku.com)
- gitlab - репозитарий (gitlab.com)
- postman - программа для тестирование энд-поинтов (postman.com)
- express

## Start development

1. `npm install`
2. `npm run start:dev`
